# 设计资料包 · 导读

> **这个文件夹装着开发本项目所需的全部设计文档。** 拿到它,按下面顺序读,就能从零开发出一个对标 Claude Code 的多 Agent 编排引擎(并落地到供应商调研 MVP)。

## 这个项目是什么

用 Python 从零复刻 **Claude Code 的多 Agent 编排核心机制**(核心 + 进阶),并落地到一个真实业务场景:**面向海外买家的中国供应商挖掘与画像**。

- **学习**:一套讲得清、可演示的多 Agent 自主编排引擎。
- **产品目标**:供应商调研 MVP(编排引擎的"第一张皮肤")。

## 复刻完成度

`01` 清单的 24 条:
- A+B+C(P0)做完 → 复刻**核心**(主循环/读并发写独占/子Agent递归/协调器/通信/安全带)
- D(P1)做完 → **几乎完全复刻**(流式/Fork/压缩/级联取消/评估循环)
- E 做完 → 接真实模型 + 供应商调研 MVP

明确**不**复刻:远程桥接、多种任务后端、链路追踪、实验开关等生产级外壳(见 01 末尾"范围边界")。
